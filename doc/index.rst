The ALICE Overwatch project aims to provide online monitoring and basic quality assurance using data from the
ALICE High Level Trigger.

For further information on the purpose of the project, quick start information, as well as detailed
architectural and operation details, please see the :doc:`README <about>`. For those interested in plugins,
please see the page on :doc:`detector plugins and trending <detectorPluginsReadme>`.

**Documentation of the Overwatch modules** is available on the :doc:`detailed code documentation
<documentation>` page.
